Consensus board-vs-pick experiment

With ESPN pre-draft boards wired in (2004-2021), the profile_plus_consensus feature set adds consensus rank, ESPN grade, and the board-vs-pick disagreement (consensus_vs_pick) to the post-draft residual model.

Feature setMean liftMedian liftWin rateWorst yearDecision
Global profile only (default)+0.0117+0.01388 / 11-0.0328Default public board
Global profile + consensus+0.0076+0.01608 / 11-0.0583Not promoted: better median, worse mean and worst year

The "player fell relative to consensus" signal helps typical years but amplifies the bad ones (2011 again). It stays experimental until it wins on all four gate metrics.

Decision: public board = profile raw APEX. Production, consensus features, and APEX+ stay experimental. Position-specific profile-only is the top challenger but still fails the strict worst-window gate.
